# 【项目企划书】《宗门裁员，靠绝色徒弟搞钱帮我续命》

## 一、 基础信息 (Basic Info)

* **书名方案（15字内）：**
* **方案A（首选）：** **宗门裁员，靠绝色徒弟搞钱帮我续命**
* **方案B（搞怪）：** **为师嫌命短，绝色孽徒疯狂内卷搞钱**
* **方案C（修罗场）：** **师尊快死了，女徒弟为搞钱打起来了**


* **类型：** 玄幻 / 系统 / 迪化流 / 搞笑 / 公路片模式
* **一句话梗概：** 一个被裁员的濒死真仙，为了填饱体内吞噬生命的“无底洞”，带着三个画风跑偏的绝色女徒弟，化身修仙界“赏金猎人团”，所过之处寸草不生、坐地分赃的爆笑故事。
* **核心基调：** **全员财迷 + 轻松反套路 + 师徒修罗场**

---

## 二、 核心卖点 (Selling Points)

1. **修仙界的“全职猎人”/“零元购”：**
* 摒弃传统的“苦修升级”和“宗门建设”，主打**“游击抢资源”**。
* 大比是为了奖金，探秘是为了抠地砖，打仗是为了卖药。一切向钱看，这种“不仅要赢，还要赚”的市侩感非常有代入感。


2. **“濒死守财奴”主角：**
* 主角的无敌不是为了装逼，是**“因为太贵了不想用，但为了救徒弟/抢更贵的宝贝不得不含泪氪金”**。这种反差极具喜感。


3. **极致的“内卷搞钱”修罗场：**
* 徒弟们争宠的方式变了：不再是比谁穿得少，而是**比谁抢得多**！
* “师尊，大师姐抢了一个宗门，我把他们祖坟挖了，我是不是更棒？” —— 这种带偏的画风是本书最大爽点。


4. **职场与现实梗的解构：**
* 队伍是“草台班子”，行动是“团建”，分赃是“发年终奖”。



---

## 三、 世界观与金手指 (World & System)

### 1. 主角身体状况：【真仙漏斗】

* **真相：** 主角是失忆真仙，体内寄生着一颗**“混元珠”**（无底洞）。
* **状态：**
* 珠子每时每刻都在吞噬灵力。
* 灵力一旦断供，立刻吞噬主角的生命（器官衰竭/咳血）。
* 主角为了活命，必须维持高强度的**“资源摄入”**。



### 2. 金手指：【薪火相传续命系统】

* **核心机制：** **资源转化 + 情绪暴击**。
* **转化逻辑：**
* **第一步（搞钱）：** 徒弟抢来的灵石/天材地宝 -> 投喂给系统 -> 转化为【灵力库存】。
* **第二步（暴击）：** 徒弟上交资源时的**心情/对师尊的爱意**决定转化倍率。
* *心情平淡（例行公事）：* **1倍**（勉强够扣）。
* *心情激动（师尊夸我了）：* **10倍**（小赚）。
* *自我攻略（这是给师尊的聘礼）：* **100倍~1000倍**（一夜暴富，主角获得【真仙一击】的使用权）。

## 四、 人设小传 (Characters)

**团队名号：** “逍遥宗” （实为修仙界第一强盗团/打工天团）

### 1. 队长/大脑：沈渊（主角）

* **人设标签：** **濒死真仙 / 极致抠门 / 战术大师**。
* **性格：** 乐天派老油条。看似贪财如命，实则为了活命。嘴上说着“我不行了”，为了徒弟敢烧干最后一滴血。
* **行动准则：** **“性价比”**。不打没收益的架，不救没钱给的人。
* **口头禅：** “这招要花我三个月寿命，得加钱！”、“徒儿们，把地砖也撬走，那是玄铁的！”

### 2. 打手/先锋：苏冷月（大徒弟）

* **人设标签：** **正道弃徒 / 伟光正的强盗 / 铁憨憨**。
* **性格：** 高冷，一根筋，逻辑自洽的武痴。
* **搞钱方式：** **【硬抢/踢馆】**。认为师尊带她抢劫是“替天行道”、“劫富济贫”（济师尊这个贫）。
* **内卷宣言：** “师尊，那宗门的门主被我砍了，库房钥匙在此！”

### 3. 刺客/军师：白怜（二徒弟）

* **人设标签：** **魔门卧底 / 腹黑绿茶 / 黑吃黑专家**。
* **性格：** 阴险，病娇，擅长伪装和下毒。
* **搞钱方式：** **【偷窃/诈骗/仙人跳】**。最懂沈渊的心思，经常背着师姐干脏活累活。
* **内卷宣言：** “师姐只会打打杀杀，不像我，直接把他们宗主策反了，连人带钱都是师尊的~”

### 4. 雷达/吉祥物：敖糯糯（三徒弟）

* **人设标签：** **龙族出逃公主 / 欧皇 / 人形聚宝盆**。
* **性格：** 社恐，爱哭，运气S级。
* **搞钱方式：** **【捡漏/碰瓷】**。走路被极品灵石绊倒，随便买的破烂里面有神器。
* **内卷宣言：** “呜呜呜，师尊，我迷路掉进一个山洞，捡了一堆亮晶晶的石头（神级灵髓），能不能抵饭钱？”

## 五、 全书主线 (Main Storyline)

* **主线任务：** **【活下去】**。为了填满体内珠子，沈渊带着徒弟踏遍九天十地，搜刮一切资源。
* **剧情模式：** **【地图单元剧】**。每到一个新地图 -> 发现肥羊（资源点） -> 师徒分工搞钱 -> 遭遇危机 -> 主角氪命装逼 -> 满载而归（分赃）。
* **终极目标：** 珠子吃饱后吐出真仙记忆，揭开主角被贬下凡的真相，最后带着三个富婆徒弟杀回上界。

## 六、 卷纲规划 (Volume Outlines)

### 第一卷：【由于被裁员，只能带徒弟去零元购了】（约15-18万字）

**核心目标：** 组建“搞钱小队”，立住“贪财”人设，完成第一波原始积累（打脸前东家）。

* **1-15章（组队篇）：**
* 沈渊被裁员，生命倒计时。
* 忽悠大徒弟苏冷月，教唆她去打劫前宗门外门弟子的储物袋（美其名曰“拿回遣散费”）。
* **爽点：** 第一次分赃，触发【100倍心情】，主角回血，确立“抢劫能致富”的方针。


* **16-40章（扩张篇）：**
* 大徒弟抢得不够多，沈渊盯上了魔门肥羊。
* 收编二徒弟白怜（原本是来杀他的，结果被沈渊的“真仙底蕴”忽悠瘸了，以为跟着沈渊能发大财）。
* **修罗场：** 两人为了“谁抢回来的灵石成色好”开始内卷。


* **41-60章（完全体）：**
* 师徒探索小秘境“进货”。捡到迷路的三徒弟敖糯糯。
* 敖糯糯随手捡宝的能力震惊全队，被确立为“团队吉祥物”。


* **61-80章（大高潮·宗门大比）：**
* 前宗门举办大比，奖金丰厚。沈渊带队报名，目标：**包揽前三名，搬空奖池**。
* **操作：** 大徒弟打擂台，二徒弟开外围盘口割韭菜，三徒弟去后山偷家。
* **结局：** 前宗主想赖账，沈渊含泪动用辛苦攒下的灵力，**一指镇压**。搬空前宗门宝库，扬长而去。



### 第二卷：【修仙界最大的“蝗虫”过境】（约20-25万字）

**核心目标：** 地图换到更广阔的修仙界，搞钱手段升级（从抢劫升级为战争财/拆迁队）。

* **剧情单元A：【遗迹拆迁队】**
* 上古遗迹开启。别的大能抢传承，沈渊带徒弟**拆房子**。
* 大徒弟拆柱子，二徒弟刮地皮，三徒弟找暗格。
* 最后发现他们拆下来的破烂拼起来才是真正的神器。


* **剧情单元B：【正魔战争财】**
* 正魔大战爆发。沈渊两头吃。
* 左手卖正道“极品金疮药”（其实是兑水的），右手卖魔道“狂暴丹”（其实是有副作用的）。
* **危机：** 玩脱了，被双方集火。
* **装逼：** 沈渊：“本来只想赚点小钱，是你们逼我破产的！” 氪掉一半寿命，强行止战，逼迫双方赔偿“精神损失费”。


* **感情线进展：**
* 徒弟们发现师尊“越有钱越虚弱”（因为钱都喂球了，没喂身体），误以为师尊得了**“散财童子绝症”**（不花钱就会死）。
* 于是徒弟们搞钱更疯了，甚至开始试图把自己“打包卖”给师尊。

### 七、 写作Tips（本地保存备注）

1. **关于“资源”的描写：** 不要只写“获得宝物”，要写**“估价”**。
* *例：* 看到神兽，第一反应不是“好强”，而是“这皮能卖5000，骨头能卖3000，抓住它！”


2. **关于“主角出手”：** 每次出手前必须有一段**“肉疼计算”**。
* *例：* “这一拳下去，我昨天刚抢的一条灵脉就没了！心好痛！”


3. **关于“反套路”：** 当读者以为要热血战斗时，一定要转折到**“坑蒙拐骗”**上；当读者以为主角要贪生怕死时，一定要转折到**“为了护犊子瞬间真仙降临”**上。
